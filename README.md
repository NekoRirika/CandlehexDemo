# CandlehexDemo

### 简介：

本项目为腾讯星跃实战训练营的怪谈主题游戏demo制作项目，持续时间从26年2月至4月，团队成员共五人，职责分工大致为两个美术，一个关卡，两个程序兼玩法兼叙事，本人负责部分程序、玩法和叙事的设计与实现。这个项目真的给了我很多收获，团队里的大家来自全国各地的高校，每个人都在努力，产出自己的创意和实现，真的非常酷=w=
